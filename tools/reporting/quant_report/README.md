# QTrading Report (Streamlit)

This folder contains a lightweight reporting app for QTrading logs stored as `.arrow` files.

Quick start (Conda)
- Open PowerShell.
- `cd` into this folder.
- Run `./setup_conda.ps1` to create or update the `Quant` environment.
- Run `./run_streamlit.ps1`.

Notes
- The app reads from a logs directory. Default is `..\..\logs` (repo root `logs/`).
- You can change the path in the sidebar.
